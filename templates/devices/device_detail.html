On the computer you want to monitor:

Option A: Automatic Setup (Easiest)
  1. Extract the downloaded ZIP file
  2. Double-click EASY_INSTALL.bat
  3. When Notepad opens, paste your token where it says PASTE_TOKEN_HERE
  4. Change the server URL to: {{ server_url }}
    Important: Use the URL shown above ({{ server_url }}). Do NOT use ws://localhost:8000 unless you're testing on this server computer itself!
  5. Save (Ctrl+S) and close Notepad
  6. Double-click START_AGENT.bat
Done! Keep the black window open. The device will show as "Online" here within 5 seconds.
Option B: Manual Setup (for advanced users)

Edit agent_config.json manually:

{
    "server_url": "{{ server_url }}",
    "device_token": "{{ device.unique_token }}",
    "device_name": "{{ device.device_name }}",
    "capture_interval": 0.5,
    "jpeg_quality": 60,
    "screen_width": 1280
}

Then run:

python agent.py
