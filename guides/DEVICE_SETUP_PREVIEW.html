Follow these 3 simple steps to start monitoring this device:

Download the monitoring agent to the computer you want to monitor.

3
Configure and Start the Agent

On the computer you want to monitor:

Option A: Automatic Setup (Easiest)
  1. Extract the downloaded ZIP file
  2. Double-click EASY_INSTALL.bat
  3. When Notepad opens, paste your token where it says PASTE_TOKEN_HERE
  4. Change the server URL to your company's hosted URL (e.g., ws://192.168.1.100:8000 or wss://controlhub.yourcompany.com)
  5. ⚠️ Important: Do NOT use localhost - use the actual server address!
  6. Save (Ctrl+S) and close Notepad
  7. Double-click START_AGENT.bat
Done! Keep the black window open. The device will show as "Online" here within 5 seconds.
