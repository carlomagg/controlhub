## 🎯 What is ControlHub Live?

ControlHub Live is a **professional remote monitoring platform** that lets you:

- 👁️ **View live screens** of connected computers in real-time
- 💬 **Send instant messages** to monitored devices
- 👥 **Manage multiple users** with role-based access
- 🌐 **Work over internet** or local network
- 📱 **Access from mobile** devices

---

## 🚀 Quick Start (5 Minutes)

### Step 1: Install ControlHub Live

```bash
# Windows
setup.bat

# Linux/macOS
chmod +x setup.sh && ./setup.sh
```

### Step 2: Start the Server

```bash
start_server.bat
```

### Step 3: Access Dashboard

Open browser: `http://localhost:8000`

**Login:**
- Username: `carlomagg`
- Password: `School67@`

### Step 4: Add Your First Device

1. Click **"Devices"** → **"Add Device"**
2. Enter device name (e.g., "My Computer")
3. Click **"Create Device"**
4. You'll see a **visual setup guide** with:
   - Download button
   - Your device token
   - Step-by-step instructions

### Step 5: Connect the Agent

1. Click **"Download Agent"** button
2. Extract the ZIP file
3. Double-click `EASY_INSTALL.bat`
4. Paste your token when Notepad opens
5. Save and close
6. Double-click `START_AGENT.bat`

**Done!** Your device will show as "Online" in 5 seconds! ✅

## 🔧 Troubleshooting

### Device Won't Connect
1. Check server is running
2. Verify token is correct
3. Check firewall settings
4. See **[COMPLETE_GUIDE.md](COMPLETE_GUIDE.md)** (Troubleshooting)

### Permission Denied
```bash
UNLOCK_ADMIN_DASHBOARD.bat
```
Then restart server

### Can't See Screen
1. Wait 5-10 seconds
2. Refresh browser
3. Check agent is running
4. Check agent logs
